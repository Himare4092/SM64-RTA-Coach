# SM64 RTA Coach

スーパーマリオ64 RTA の通し走行を AI が解析し、**「良い点(続けるべきこと)」と「改善点」を日本語でまとめてくれる** Windows 用デスクトップアプリです。

対応カテゴリ:

- **メインカテゴリー**: 0 Star / 1 Star / 16 Star / 70 Star / 120 Star
- **サブカテゴリー**: 15 Star / 16 Star NO LBLJ / 30 Star / 31 Star / 45 Star / 50 Star / 90 Star

## 何ができるの?

1. 通し走行の**録画ファイル**と、LiveSplit の**スプリットファイル(.lss)** を読み込む
2. 完走した試行を選ぶと、区間タイム(今回 / PB / 区間ベスト)を自動で集計
3. 各区間の終盤(スター取得直前)の画面を動画から自動で切り出し(最大18枚)
4. 区間タイム表 + スクリーンショット + カテゴリの参考タイムを AI(Claude)が分析し、以下の構成で講評をリアルタイム表示:
   - **総評**
   - **良い点(続けるべきこと)** — 金に近い区間・PB更新区間を根拠つきで
   - **改善点(優先度順)** — 何がどれくらい遅いか → 原因の推測 → 具体的な練習方法
   - **区間別コメント**
   - **次の練習メニュー**
5. 解析結果はローカルに保存され、次回以降は**過去の自分の走りとの伸び**も講評に反映されます

## 動作環境・必要なもの

- Windows 10 / 11(64bit)
- **Anthropic API キー**([console.anthropic.com](https://console.anthropic.com/) で取得)
  - 解析には Claude API の従量課金が発生します。1回の解析はおおむね数十円程度が目安です(画像枚数・講評の長さ・為替により変動)
- LiveSplit で計測した `.lss` ファイル(完走後に保存したもの)
- 通し走行の動画ファイル(mp4 / mkv / avi / mov / webm など)

ffmpeg はアプリに同梱されているため、別途インストールは不要です。

## 使い方

1. 右上の **⚙ 設定** から Anthropic API キーを保存する(環境変数 `ANTHROPIC_API_KEY` でも可)
2. **動画ファイル**と **.lss** を選択する
3. 解析したい**試行**を選ぶ(完走した試行のみ・新しい順に表示されます)
4. **カテゴリ**を確認する(.lss のカテゴリ名から自動推測されます)
5. **タイマー始動位置のオフセット**を入力する(下記参照)
6. **★ AI解析を開始** を押す — 講評がリアルタイムで流れてきます
7. 過去の講評は「解析履歴」タブからいつでも見返せます

### タイマー始動位置のオフセットについて

「動画の何秒地点でタイマーが動き始めるか」を入力してください(録画開始とタイマー始動がずれている場合に必要です)。

SM64 RTA の計測ルールに従い、**タイマーは 0:00 ではなく 1.36秒表示から始動する**前提で動画時刻を計算します(動画時刻 = 区間の累積タイム − 1.36 + オフセット)。ここが正確だと、スクリーンショットが「各スター取得の直前」を正しく捉え、講評の精度が上がります。

## プライバシー / 送信されるデータ

- **動画ファイルそのものは外部にアップロードされません。** フレーム抽出はすべてローカルの ffmpeg で行われます
- Anthropic API に送信されるのは「区間タイムなどの数値データ」と「抽出した静止画(横640px・最大18枚)」のみです
- API キーと解析履歴は自分の PC 内(`%APPDATA%\sm64-rta-coach`)にのみ保存されます

### 技術構成

- Electron + React + TypeScript(electron-vite)
- フレーム抽出: ffmpeg(ffmpeg-static 同梱)
- .lss 解析: fast-xml-parser
- AI講評: Anthropic Claude API(claude-opus-4-8・ストリーミング)

## 注意事項

- 本アプリは非公式のファンメイドツールです。任天堂および Anthropic とは関係ありません
- アプリ内蔵の「参考タイム」はあくまで大まかな目安です。講評は走者本人の PB・区間ベスト・過去走行との比較を優先する設計になっています
